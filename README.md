## 👨‍💻 My projects
* [Random Video Chat Web](https://github.com/shashwatiwari/Random-Video-Chat) - Innovative platform made using <strong>React Js</strong> and <strong>WebRtc</strong> allows spontaneous conversations by pairing users randomly for video
chats.
* [Online Hotel Booking Web](https://github.com/shashwatiwari/Hotel-Booking) - A Full Stack Web Application made using <strong>NodeJS</strong>, <strong>React Js</strong> and <strong>Tailwind CSS</strong> ensuring a user-friendly interface and seamless
booking experience for guests.
* [Hostel Management System](https://github.com/shashwatiwari/Hostel_Management_System) - A comprehensive hostel management system using <strong>NodeJS</strong>, <strong>React Js</strong> and <strong>MySql</strong> to automate and streamline hostel operations.
